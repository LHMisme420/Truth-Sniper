# TruthSniper

The world didn't know it needed this... until now.

A real-time bullshit detector for X/Twitter.  
Paste any tweet → get instant fact-checks with sources → watch lies die in 4K.

Built in pure Python. No mercy. No corporate funding. Just truth.

## Features (MVP v0.2 - Dec 2025)
- Fetches any public tweet/thread (no auth needed)
- Extracts factual claims automatically
- Real-time fact-checking (upgradable to Grok, Perplexity, or local LLM)
- Beautiful terminal UI with Rich
- One day: browser extension + auto-reply mode

## Install & Run (30 seconds)

```bash
git clone https://github.com/badasspython/truthsniper.git
cd truthsniper
pip install -r requirements.txt
python main.py
